🛠️ КАК ПОДГОТОВИТЬ ИЗОБРАЖЕНИЯ:

ВАРИАНТ 1: Онлайн редактор
1. Открой https://www.photopea.com/
2. Загрузи изображение 123.png
3. Вырежи каждый эмодзи
4. Измени размер: Image → Image Size → 128x128
5. Сохрани как PNG

ВАРИАНТ 2: Paint.NET / GIMP
1. Открой изображение
2. Выдели каждый эмодзи (прямоугольное выделение)
3. Скопируй в новый файл
4. Измени размер до 128x128
5. Сохрани как PNG

ВАРИАНТ 3: Photoshop
1. Открой 123.png
2. Используй Slice Tool для нарезки
3. Экспортируй каждый слайс
4. Resize до 128x128
5. Save for Web (PNG-24)
