# 🚀 НАЧНИ ЗДЕСЬ

## Что это?

Discord бот TTFD с улучшениями:
- ✅ Slash-команды (/) и обычные (!)
- ✅ Прогресс-бары
- ✅ Daily streak бонусы
- ✅ Мини-игры
- ✅ Красивые embed'ы

## Быстрый старт

### 1. Запусти бота

Двойной клик на:
```
ЗАПУСК_УЛУЧШЕННЫЙ.bat
```

Или:
```bash
python main.py
```

### 2. Попробуй команды

В Discord напиши:
```
/profile
/daily
/dice
```

Или с префиксом:
```
!profile
!daily
!slut
```

### 3. Готово! 🎉

## Основные команды

```
/profile    - твой профиль с прогресс-баром
/daily      - ежедневная награда + бонус за серию
/rank       - твой ранг
/top        - лидеры
/dice       - кубик (1-6)
/coinflip   - монетка
```

## Проблемы?

1. **Slash-команды не видны?**
   → Перезапусти Discord клиент

2. **Rate limit?**
   → Подожди 30-60 минут

3. **Другие проблемы?**
   → Читай `ТЕСТИРОВАНИЕ.md`
