# 📁 Структура проекта TTFD-Discord

## 🗂️ Организация файлов

Проект организован по типам файлов для удобства навигации:

```
TTFD-Discord/
├── 📁 py/              # Python файлы
│   ├── bot.py          # Основной файл бота
│   ├── config.py       # Конфигурация
│   ├── database.py     # База данных
│   ├── font_converter.py
│   ├── updates_system.py
│   └── set_update.py
│
├── 📁 json/            # JSON файлы
│   ├── version.json    # Версия и история
│   ├── auto_update.json
│   └── user_data.json
│
├── 📁 md/              # Markdown документация
│   ├── README.md
│   ├── АВТООБНОВЛЕНИЕ.md
│   └── ...
│
├── 📁 txt/             # Текстовые файлы
│   └── ...
│
├── 📁 bat/             # Batch скрипты
│   └── ЗАПУСК_УЛУЧШЕННЫЙ.bat
│
├── 📁 шрифт/           # Шрифты бота
│   └── шрифт бота.txt
│
├── ЗАПУСК.bat          # Быстрый запуск
├── УСТАНОВИТЬ_ОБНОВЛЕНИЕ.bat
├── .env                # Переменные окружения
└── .gitignore
```

---

## 🚀 Запуск бота

### Способ 1: Через bat файл (рекомендуется)
Запустите файл в корне:
```
ЗАПУСК.bat
```

### Способ 2: Через командную строку
```bash
python py/bot.py
```

---

## 🔄 Установка автообновления

### Способ 1: Через bat файл
Запустите:
```
УСТАНОВИТЬ_ОБНОВЛЕНИЕ.bat
```

### Способ 2: Через командную строку
```bash
python py/set_update.py "Изменение 1 | Изменение 2"
```

---

## 📂 Описание папок

### py/
Все Python модули и скрипты:
- `bot.py` - главный файл бота
- `config.py` - настройки (токены, ID каналов)
- `database.py` - работа с базой данных
- `updates_system.py` - система версионирования
- `set_update.py` - установка автообновления

### json/
Все JSON файлы с данными:
- `version.json` - текущая версия и история обновлений
- `auto_update.json` - настройки автообновления
- `user_data.json` - данные пользователей

### md/
Вся документация в формате Markdown:
- Инструкции
- Руководства
- Описания систем

### bat/
Batch скрипты для Windows:
- Скрипты запуска
- Утилиты

### txt/
Текстовые файлы:
- Конфигурации
- Заметки

---

## ⚙️ Важные файлы

### В корне проекта:
- `.env` - переменные окружения (токены, пароли)
- `.gitignore` - игнорируемые файлы для Git
- `ЗАПУСК.bat` - быстрый запуск бота
- `УСТАНОВИТЬ_ОБНОВЛЕНИЕ.bat` - установка автообновления

---

## 🔧 Настройка

1. Скопируйте `.env.example` в `.env`
2. Заполните токены и настройки в `.env`
3. Запустите `ЗАПУСК.bat`

---

## 📝 Примечания

- Все пути в коде обновлены для работы с новой структурой
- Бот запускается из папки `py/`
- JSON файлы читаются из папки `json/`
- Документация находится в папке `md/`
