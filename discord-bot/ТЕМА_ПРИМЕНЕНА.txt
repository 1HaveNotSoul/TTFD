🎨 ЦВЕТОВАЯ ПАЛИТРА:

Чёрно-красный градиент (8 оттенков):
1. #0d0d0d - Чёрный
2. #1a0000 - Очень тёмный красный
3. #330000 - Тёмный красный
4. #660000 - Средне-тёмный красный
5. #8b0000 - Кроваво-красный ⭐ (основной для игр)
6. #cc0000 - Красный
7. #ff0000 - Ярко-красный ⭐ (основной для магазина)
8. #ff4444 - Светло-красный ⭐ (основной для профиля)

Специальные цвета:
- Успех: #00ff00 (зелёный)
- Ошибка: #ff0000 (красный)
- Предупреждение: #ff6600 (оранжево-красный)
- Информация: #ff4444 (светло-красный)

📝 КАК ИСПОЛЬЗОВАТЬ ТЕМУ:

1. Импорт (уже добавлен в bot.py):
   from theme import BotTheme, game_embed, profile_embed, success_embed, error_embed, warning_embed

2. Быстрые функции:
   
   Игры:
   embed = game_embed(
       title=convert_to_font("🎮 название игры"),
       description=convert_to_font("описание")
   )
   
   Профиль:
   embed = profile_embed(
       title=convert_to_font("👤 профиль")
   )
   
   Успех:
   embed = success_embed(
       title=convert_to_font("✅ успех!")
   )
   
   Ошибка:
   embed = error_embed(
       title=convert_to_font("❌ ошибка!")
   )
   
   Предупреждение:
   embed = warning_embed(
       title=convert_to_font("⚠️ внимание!")
   )

3. Универсальный способ:
   embed = BotTheme.create_embed(
       title=convert_to_font("заголовок"),
       embed_type='game'  # или 'profile', 'shop', 'rank', 'ticket', 'music'
   )
