🚀 ЧТО ДЕЛАТЬ ДАЛЬШЕ:

1️⃣ ЗАГРУЗИТЬ ЭМОДЗИ НА DISCORD СЕРВЕР
   
   Открой Discord → Настройки сервера → Эмодзи → Загрузить эмодзи
   
   Загрузи каждый файл с названием:
   ┌─────────────────────────────────────┐
   │ F.png → Название: rank_f            │
   │ E.png → Название: rank_e            │
   │ D.png → Название: rank_d            │
   │ C.png → Название: rank_c            │
   │ B.png → Название: rank_b            │
   │ A.png → Название: rank_a            │
   │ S.png → Название: rank_s            │
   └─────────────────────────────────────┘

2️⃣ ПОЛУЧИТЬ ID ЭМОДЗИ
   
   В любом канале Discord напиши:
   \:rank_f:
   
   Отправь → получишь:
   <:rank_f:1234567890123456789>
   
   Скопируй ВЕСЬ код для каждого эмодзи:
   \:rank_f: → <:rank_f:ID>
   \:rank_e: → <:rank_e:ID>
   \:rank_d: → <:rank_d:ID>
   \:rank_c: → <:rank_c:ID>
   \:rank_b: → <:rank_b:ID>
   \:rank_a: → <:rank_a:ID>
   \:rank_s: → <:rank_s:ID>

3️⃣ ОТПРАВИТЬ МНЕ ID
   
   Скопируй все 7 строк и отправь мне:
   
   Пример:
   <:rank_f:1234567890123456789>
   <:rank_e:2345678901234567890>
   <:rank_d:3456789012345678901>
   <:rank_c:4567890123456789012>
   <:rank_b:5678901234567890123>
   <:rank_a:6789012345678901234>
   <:rank_s:7890123456789012345>
