🔍 КЛЮЧЕВЫЕ СТРОКИ ДЛЯ ПОИСКА:
-----------------------------

1️⃣ КОЛИЧЕСТВО КОМАНД В BOT.TREE:
   Ищи строку: "🔍 DEBUG: Команд в bot.tree:"
   
   Ожидаемый результат: 30 команд (27 + 3)
   
   ❓ Если 0 команд:
      → Команды не регистрируются в bot.tree
      → Проблема в функциях setup_slash_commands() и setup_game_commands()
   
   ✅ Если 30 команд:
      → Команды регистрируются правильно
      → Проблема в sync или GUILD_ID

2️⃣ СПИСОК КОМАНД:
   Ищи строку: "🔍 DEBUG: Список команд:"
   
   Должны быть:
   - profile, balance, rank, top, shop, inventory
   - daily, work, clear, help, ping, stats
   - links, buy, pay, dice, coinflip
   - ticket, close
   - updatecommands, setupverification, setuptickets
   - setuprankroles, syncrankroles
   - gamelink, unlink, gamestats
   
   Всего: 30 команд

3️⃣ GUILD_ID:
   Ищи строку: "🔍 DEBUG: GUILD_ID ="
   
   Должно быть: 1466282399540396032
   
   ❌ Если 0 или другое число:
      → Проблема в .env файле
      → GUILD_ID не установлен или неправильный

4️⃣ GUILD SYNC:
   Ищи строку: "✅ Синхронизировано X slash команд с Discord (guild sync)"
   
   ❓ Если 0 команд:
      → Guild sync не работает
      → Смотри следующий пункт (global sync)
   
   ✅ Если 30 команд:
      → Всё работает идеально!
      → Команды должны появиться мгновенно

5️⃣ GLOBAL SYNC (если guild sync вернул 0):
   Ищи строку: "⚠️ WARNING: Синхронизировано 0 команд!"
   Затем: "✅ Global sync: X команд"
   
   ✅ Если global sync вернул 30:
      → Команды зарегистрированы глобально
      → Появятся через 1 час
      → Можно использовать global sync вместо guild

📊 СЦЕНАРИИ И РЕШЕНИЯ:
---------------------

СЦЕНАРИЙ A: bot.tree = 0, guild sync = 0
   ПРИЧИНА: Команды не регистрируются
   РЕШЕНИЕ: Изменить способ регистрации команд

СЦЕНАРИЙ B: bot.tree = 30, guild sync = 0, global sync = 30
   ПРИЧИНА: Guild sync не работает, но global работает
   РЕШЕНИЕ: Использовать global sync (команды через 1 час)

СЦЕНАРИЙ C: bot.tree = 30, guild sync = 30
   ПРИЧИНА: Всё работает!
   РЕШЕНИЕ: Проверить команды в Discord

СЦЕНАРИЙ D: Команды видны в Discord, но не выполняются
   ПРИЧИНА: Старые команды из предыдущего деплоя
   РЕШЕНИЕ: Очистить команды и пересинхронизировать
