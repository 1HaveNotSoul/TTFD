ШАГ 1: ОРГАНИЗАЦИЯ ПРОЕКТОВ
────────────────────────────
Текущая папка: C:\Users\brawl\OneDrive\Desktop\папки\TTFD

Запустите:
   cd C:\Users\brawl\OneDrive\Desktop\папки\TTFD
   .\ОРГАНИЗОВАТЬ_ПРОЕКТЫ.bat

Это скопирует:
   TTFD-Website  → TTFD/website/
   TTFD-Discord  → TTFD/discord-bot/
   TTFD-Cleaner  → TTFD/cleaner/

Время: ~2 минуты


ШАГ 2: ИНИЦИАЛИЗАЦИЯ GIT
────────────────────────────
   cd TTFD
   git init
   git branch -M main
   git add .
   git commit -m "Initial commit: TTFD unified project"

Время: ~1 минута


ШАГ 3: СОЗДАНИЕ GITHUB РЕПОЗИТОРИЯ
────────────────────────────
1. Откройте: https://github.com/new
2. Название: TTFD
3. Description: The First Descendants - Unified Project
4. Visibility: Public или Private
5. НЕ добавляйте README, .gitignore, license
6. Create repository

Время: ~2 минуты


ШАГ 4: ПОДКЛЮЧЕНИЕ К GITHUB
────────────────────────────
Скопируйте команды с GitHub:

   git remote add origin https://github.com/yourusername/TTFD.git
   git push -u origin main

Время: ~1 минута


ШАГ 5: ПОДГОТОВКА К ДЕПЛОЮ
────────────────────────────
1. Зарегистрируйтесь на https://render.com
2. Подключите GitHub репозиторий
3. Получите Discord OAuth данные:
   - https://discord.com/developers/applications
   - Client ID и Client Secret
4. Получите Discord Bot Token:
   - Bot → Reset Token

Время: ~10 минут


ШАГ 6: ДЕПЛОЙ POSTGRESQL
────────────────────────────
Render Dashboard → New → PostgreSQL

Настройки:
   Name: ttfd-database
   Database: ttfd
   User: ttfd_user
   Region: Frankfurt (EU Central)
   Plan: Free

Скопируйте Internal Database URL!

Время: ~5 минут


ШАГ 7: ДЕПЛОЙ WEBSITE
────────────────────────────
Render Dashboard → New → Web Service

Настройки:
   Name: ttfd-website
   Repository: TTFD
   Root Directory: website
   Build Command: pip install -r requirements.txt
   Start Command: python main.py

Environment Variables:
   SECRET_KEY=<сгенерируйте>
   PORT=10000
   DISCORD_CLIENT_ID=<из Discord Portal>
   DISCORD_CLIENT_SECRET=<из Discord Portal>
   DISCORD_REDIRECT_URI=https://ttfd-website.onrender.com/auth/discord/callback
   DATABASE_URL=<из PostgreSQL>

Время: ~10 минут


ШАГ 8: ДЕПЛОЙ DISCORD BOT
────────────────────────────
Render Dashboard → New → Background Worker

Настройки:
   Name: ttfd-discord-bot
   Repository: TTFD
   Root Directory: discord-bot
   Build Command: pip install -r requirements.txt
   Start Command: python main.py

Environment Variables:
   DISCORD_TOKEN=<из Discord Portal>
   GUILD_ID=<ваш Server ID>
   DATABASE_URL=<из PostgreSQL>

Время: ~10 минут


ШАГ 9: ОБНОВЛЕНИЕ OAUTH REDIRECT
────────────────────────────
После деплоя Website:

1. Скопируйте URL: https://ttfd-website.onrender.com
2. Discord Portal → OAuth2 → Redirects
3. Добавьте: https://ttfd-website.onrender.com/auth/discord/callback
4. Save Changes

Время: ~2 минуты


ШАГ 10: ПРОВЕРКА РАБОТЫ
────────────────────────────
Website:
   1. Откройте URL сайта
   2. Попробуйте войти через Discord
   3. Проверьте профиль

Discord Bot:
   1. Проверьте что бот онлайн
   2. Попробуйте команды: !ping, !profile

Cleaner:
   1. Уже готов к распространению
   2. Файл: cleaner/dist/TTFD-Cleaner-Menu.exe

Время: ~5 минут

ОБЩЕЕ ВРЕМЯ: ~50 минут
