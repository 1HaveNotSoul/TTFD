# 🏠 Запуск Discord Бота Локально

## Шаг 1: Настрой .env файл

Открой файл `TTFD-Website/.env` и замени значения на реальные:

```env
# Discord Bot Token (из Discord Developer Portal)
DISCORD_TOKEN=твой_реальный_токен_бота

# ID твоего Discord сервера
GUILD_ID=твой_guild_id

# Порт для локального веб-сервера
WEB_PORT=5000

# Секретный ключ (любая случайная строка)
SECRET_KEY=local_dev_secret_key_12345

# Discord OAuth (если нужна авторизация)
DISCORD_CLIENT_ID=твой_client_id
DISCORD_CLIENT_SECRET=твой_client_secret
DISCORD_REDIRECT_URI=http://localhost:5000/auth/discord/callback
```

### Где взять данные:

1. **DISCORD_TOKEN**: 
   - Открой https://discord.com/developers/applications
   - Выбери своё приложение
   - Bot → Reset Token → Скопируй токен

2. **GUILD_ID**:
   - В Discord включи Developer Mode (Settings → Advanced → Developer Mode)
   - Правый клик на свой сервер → Copy Server ID

3. **DISCORD_CLIENT_ID** и **DISCORD_CLIENT_SECRET**:
   - В Discord Developer Portal → OAuth2 → General
   - Client ID и Client Secret

---

## Шаг 2: Установи зависимости

Открой терминал в папке `TTFD-Website` и выполни:

```bash
pip install -r requirements.txt
```

Или если используешь виртуальное окружение:

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

---

## Шаг 3: Запусти бота

### Вариант A: Только бот (без веб-сервера)

```bash
python bot.py
```

### Вариант B: Бот + Веб-сервер вместе

```bash
python main.py
```

**ВАЖНО:** В `main.py` измени `BOT_ENABLED = False` на `BOT_ENABLED = True`

---

## Шаг 4: Проверь что бот работает

В терминале должно появиться:

```
✅ Бот успешно запущен!
📛 Имя: TTFD Bot
🆔 ID: 123456789
🌐 Серверов: 1
👥 Пользователей: 10
```

В Discord бот должен быть **онлайн** (зелёный статус).

- `!profile` - Твой профиль

---

## Troubleshooting:

### Ошибка: "No module named 'discord'"
```bash
pip install discord.py==2.3.2
```

### Ошибка: "Invalid token"
- Проверь что DISCORD_TOKEN правильный
- Токен должен начинаться с букв (не цифр)
- Попробуй сгенерировать новый токен

### Бот не отвечает на команды:
- Проверь что у бота есть права "Read Messages" и "Send Messages"
- Проверь что Message Content Intent включен в Developer Portal

---

## После тестирования:

Когда закончишь тестировать локально:

1. **НЕ коммить .env файл** с реальными токенами!
2. Верни `BOT_ENABLED = False` в `main.py` (для Render)
3. Бот на Render включится автоматически когда снимется rate limit
