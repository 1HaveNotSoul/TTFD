RAILWAY - НАСТРОЙКА:
═══════════════════════════════════════════════════════════════

□ Settings → Root Directory
  □ Установить: website
  □ Сохранить

□ Settings → Domains
  □ Нажать "Generate Domain"
  □ Записать домен: _______________________

□ Variables → Add Variables
  □ SECRET_KEY = (сгенерировать случайную строку)
  □ DISCORD_CLIENT_ID = (из Discord Developer Portal)
  □ DISCORD_CLIENT_SECRET = (из Discord Developer Portal)
  □ DISCORD_REDIRECT_URI = https://[твой-домен].railway.app/auth/discord/callback
  □ PORT = 10000

□ Закоммитить и запушить
