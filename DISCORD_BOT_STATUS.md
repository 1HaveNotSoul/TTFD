Открой [Render Dashboard](https://dashboard.render.com) и проверь:

- **Текущий коммит**: Должен быть `1f61b37` или новее
- **Если развернут старый коммит** (3c98a9d): Нажми **"Manual Deploy"** → **"Deploy latest commit"**

### 2. Проверить Environment Variables

В Render Dashboard → твой сервис → Environment → убедись что установлены:

```
DISCORD_TOKEN=твой_токен_бота
GUILD_ID=ID_твоего_Discord_сервера
SECRET_KEY=любая_случайная_строка
DISCORD_CLIENT_ID=для_OAuth
DISCORD_CLIENT_SECRET=для_OAuth
DISCORD_REDIRECT_URI=https://ttfd.onrender.com/auth/discord/callback
```

### 3. Подождать Снятия Rate Limit

⏰ **Подожди 15-30 минут** после последней попытки подключения.

Discord автоматически снимет блокировку.

## 📝 Структура Проекта

### Основные Файлы:

- **`main.py`** - Точка входа, запускает бота и веб-сервер одновременно
- **`bot.py`** - Discord bot с командами (!ping, !profile, !ticket, и т.д.)
- **`app.py`** - Flask веб-сервер с Discord OAuth
- **`discord_oauth.py`** - Авторизация через Discord
- **`config.py`** - Конфигурация и переменные окружения
- **`database.py`** / **`database_postgres.py`** - База данных

### Как Работает:

1. **`main.py`** запускается на Render
2. Веб-сервер стартует в отдельном потоке (daemon)
3. Discord bot запускается в основном потоке
4. Оба работают одновременно на одном процессе

## 🎮 Команды Бота

### Основные:
- `!ping` - Проверка задержки
- `!stats` - Статистика бота
- `!link` - Ссылка на сайт
- `!slut` - Случайное фото котика 🐱

### Профиль:
- `!profile [@user]` - Профиль пользователя
- `!rank` - Текущий ранг
- `!top` - Таблица лидеров
- `!daily` - Ежедневная награда

### Поддержка:
- `!ticket` - Создать тикет
- `!close` - Закрыть тикет

### Модерация:
- `!clear <число>` - Очистить сообщения
