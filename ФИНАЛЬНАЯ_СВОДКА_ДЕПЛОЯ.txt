🔄 КАК РАБОТАЕТ АВТООБНОВЛЕНИЕ:
═══════════════════════════════════════════════════════════════

ЛОКАЛЬНО:
1. Вносишь изменения в код
2. git commit -m "feat: новая функция"
3. Git hook добавляет в auto_update.json
4. git push отправляет на GitHub

НА RAILWAY:
1. Railway получает webhook от GitHub
2. Автоматически деплоит изменения
3. Бот запускается
4. Видит auto_update.json с enabled: true
5. Отправляет обновление в Discord канал
6. Очищает auto_update.json
