ШАГ 1: ОБНОВЛЕНИЕ
═══════════════════════════════════════════════════════════════

1. Закройте TTFD-Cleaner.exe (если открыт)
2. Запустите: ОБНОВИТЬ_EXE.bat
3. Дождитесь завершения копирования

═══════════════════════════════════════════════════════════════

ШАГ 2: ЗАПУСК
═══════════════════════════════════════════════════════════════

1. Запустите TTFD-Cleaner.exe
2. Перейдите на вкладку "Автозапуск"
3. Нажмите "Сканировать"
4. Дождитесь завершения (10-30 секунд)

═══════════════════════════════════════════════════════════════

ШАГ 3: ИСПОЛЬЗОВАНИЕ
═══════════════════════════════════════════════════════════════

ФИЛЬТРЫ:
--------
☑ Показать Microsoft - только системные
☑ Показать сторонние - только сторонние программы
☑ Показать отключённые - только отключённые

ЦВЕТА:
------
🔴 Красный - защищённые (нельзя отключить)
🔵 Синий - Microsoft (системные)
⚫ Чёрный - сторонние (можно отключить)
⚪ Серый - отключённые

ДЕЙСТВИЯ:
---------
1. Выберите элемент в списке
2. Нажмите "Отключить" или "Включить"
3. Проверьте результат в логе

ЭКСПОРТ:
--------
1. Нажмите "Экспорт в TXT"
2. Выберите место сохранения
3. Откройте файл для просмотра

═══════════════════════════════════════════════════════════════

ВАЖНО:
------
❗ НЕ отключайте красные элементы (защищённые)
❗ Будьте осторожны с синими элементами (системные)
✅ Безопасно отключать чёрные элементы (сторонние)

КАТЕГОРИИ АВТОЗАПУСКА:
----------------------
✅ Logon - Run ключи, папки автозапуска
✅ Explorer - Shell folders
✅ Services - службы Windows
✅ Scheduled Tasks - планировщик заданий
✅ Drivers - драйверы ядра
✅ Winlogon - Shell, Userinit, UIHost
✅ Browser Helper Objects - BHO
✅ Codecs - кодеки
✅ Print Monitors - мониторы печати
✅ LSA Providers - провайдеры LSA (только чтение)
✅ Boot Execute - загрузочные команды (только чтение)
