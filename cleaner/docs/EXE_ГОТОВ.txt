🎯 ПЕРВЫЙ ЗАПУСК:

1. Двойной клик на TTFD-Cleaner.exe
2. Откроется окно программы
3. Создайте baseline (Отчёты → Экспорт Baseline)
4. Попробуйте очистку (Очистка → Сканировать)

═══════════════════════════════════════════════════════════════

⚠️ ВАЖНО:

1. TTFD.Cleaner.Cli.exe должен быть рядом!
   Без него программа не будет работать.

2. Для опасных категорий нужны права администратора:
   ПКМ на TTFD-Cleaner.exe → "Запуск от имени администратора"

3. Windows Defender может проверять файл при первом запуске
   (это нормально для новых .exe файлов)

📦 РАСПРОСТРАНЕНИЕ:

Чтобы передать программу другому компьютеру:

1. Скопируйте ОБА файла:
   - TTFD-Cleaner.exe
   - TTFD.Cleaner.Cli.exe

2. Положите их в одну папку

3. Запустите TTFD-Cleaner.exe

Всё! Python не нужен на целевом компьютере.
