📊 ПОТОК ДАННЫХ:

1. СКАНИРОВАНИЕ:
   GUI → scan-cleaning --categories temp,cache
   CLI → сканирует файлы → возвращает JSON
   GUI ← получает результат → отображает

2. ОЧИСТКА:
   GUI → apply-cleaning --categories temp --yes
   CLI → удаляет файлы → возвращает JSON
   GUI ← получает результат → показывает успех

3. АВТОЗАПУСК:
   GUI → list-startup
   CLI → читает Registry + Startup folder → JSON
   GUI ← получает список → отображает в таблице

4. BASELINE:
   GUI → export-baseline
   CLI → сохраняет текущее состояние → Config/baseline.json
   GUI ← получает подтверждение
