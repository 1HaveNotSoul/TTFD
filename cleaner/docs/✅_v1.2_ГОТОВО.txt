КАТЕГОРИИ АВТОЗАПУСКА (12 шт.)
═══════════════════════════════════════════════════════════════

1. Logon
   - HKLM\Software\Microsoft\Windows\CurrentVersion\Run
   - HKCU\Software\Microsoft\Windows\CurrentVersion\Run
   - Startup folders (User/All Users)

2. Explorer
   - Shell folders
   - Shell extensions

3. Services
   - Службы Windows с автозапуском
   - Automatic и Automatic (Delayed Start)

4. Scheduled Tasks
   - Задачи планировщика
   - Только активные задачи

5. Drivers
   - Драйверы ядра
   - Boot и System start

6. Winlogon
   - Shell (explorer.exe)
   - Userinit (userinit.exe)
   - UIHost

7. Browser Helper Objects (BHO)
   - Расширения браузеров
   - CLSID в реестре

8. Codecs
   - Кодеки аудио/видео
   - Фильтры DirectShow

9. Print Monitors
   - Мониторы печати
   - DLL в System32

10. LSA Providers (только чтение)
    - Security Packages
    - Authentication Packages
    - Notification Packages

11. Boot Execute (только чтение)
    - Команды при загрузке
    - Session Manager

ФУНКЦИИ
═══════════════════════════════════════════════════════════════

[Сканировать]
- Полное сканирование всех 12 категорий
- Занимает 10-30 секунд
- Отображает все элементы автозапуска

[Включить] / [Отключить]
- Управление выбранным элементом
- Защищённые элементы нельзя отключить
- Требуются права администратора для некоторых элементов

[Экспорт в TXT]
- Сохранение списка автозапуска в файл
- Группировка по категориям
- Включает пути и статусы

[Легенда]
- Объяснение цветового кодирования
- Справка по использованию
