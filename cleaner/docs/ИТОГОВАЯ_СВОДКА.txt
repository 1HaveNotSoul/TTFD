🏗️ АРХИТЕКТУРА:

GUI (Python + Tkinter):
- Главное окно 1000x650
- 6 вкладок (Notebook)
- Лог событий (ScrolledText)
- Статус-бар с информацией
- Асинхронное выполнение (threading)
- JSON обмен с CLI

Backend (C# .NET 8):
- Консольное приложение
- JSON вывод результатов
- Команды через аргументы
- Registry API
- PowerShell integration
- Standalone executable

Взаимодействие:
GUI → subprocess → CLI → JSON → GUI
