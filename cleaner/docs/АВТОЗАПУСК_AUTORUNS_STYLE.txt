🎯 КАК ИСПОЛЬЗОВАТЬ:

1. Запустите TTFD-Cleaner.exe
2. Перейдите в "Автозапуск"
3. Нажмите "Обновить"
4. Увидите древовидную структуру с категориями
5. Раскройте категорию для просмотра элементов
6. Выберите элемент и нажмите "Отключить" или "Включить"
7. Используйте фильтры для отображения нужных категорий
8. Нажмите "Экспорт в TXT" для сохранения списка

📊 КАТЕГОРИИ АВТОЗАПУСКА:

1. LOGON (Run keys):
   - HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\Run
   - HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Run
   - HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\RunOnce
   - HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\RunOnce

2. LOGON (Startup folders):
   - %APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup
   - %ProgramData%\Microsoft\Windows\Start Menu\Programs\Startup

3. EXPLORER:
   - Shell Folders
   - User Shell Folders

4. SERVICES:
   - Windows Services с автозапуском
   - Только работающие сервисы

5. SCHEDULED TASKS:
   - Задачи с триггером Logon/Startup

6. DRIVERS:
   - Драйверы ядра (Kernel drivers)
   - Только с автозапуском

7. WINLOGON:
   - Shell, Userinit, UIHost
   - Критические компоненты входа

8. BROWSER HELPER OBJECTS:
   - Расширения браузеров
   - COM объекты

9. CODECS:
   - Аудио/видео кодеки
   - Drivers32

10. PRINT MONITORS:
    - Мониторы печати
    - Системные компоненты

11. LSA PROVIDERS:
    - Security Providers
    - Read-only (только просмотр)

12. BOOT EXECUTE:
    - Команды при загрузке
    - Read-only (только просмотр)

📝 ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ:

ОТКЛЮЧИТЬ АВТОЗАПУСК ПРИЛОЖЕНИЯ:
1. Найдите приложение в списке
2. Выберите его
3. Нажмите "Отключить"
4. Приложение не будет запускаться при входе

НАЙТИ ВСЕ СЕРВИСЫ:
1. Включите фильтр "Services"
2. Отключите остальные фильтры
3. Нажмите "Обновить"
4. Увидите только сервисы

ЭКСПОРТИРОВАТЬ СПИСОК:
1. Нажмите "Экспорт в TXT"
2. Файл сохранится как startup_export_YYYYMMDD_HHMMSS.txt
3. Откройте файл для просмотра
