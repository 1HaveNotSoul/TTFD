═══════════════════════════════════════════════════════════════

📋 ЧТО ЭТО?

TTFD-Cleaner - это инструмент для безопасной очистки и оптимизации
Windows с акцентом на безопасность и прозрачность.

Возможности:
✅ Очистка временных файлов и кэша
✅ Управление автозапуском приложений
✅ Удаление ненужных UWP приложений
✅ Baseline и откат изменений
✅ Защита системных компонентов

═══════════════════════════════════════════════════════════════

🚀 БЫСТРЫЙ СТАРТ (3 ШАГА):

ШАГ 1: СБОРКА BACKEND
   Запустите: BUILD.bat
   
   Это создаст файл TTFD.Cleaner.Cli.exe

ШАГ 2: ЗАПУСК GUI
   Запустите: ЗАПУСК.bat
   
   Откроется главное окно программы

ШАГ 3: ИСПОЛЬЗОВАНИЕ
   1. Перейдите в "Отчёты" → "Экспорт Baseline" (создайте точку восстановления)
   2. Перейдите в "Очистка" → выберите категории → "Сканировать"
   3. Проверьте результаты → "Очистить"

📞 ТИПИЧНЫЕ ВОПРОСЫ:

Q: "CLI не найден"
A: Запустите BUILD.bat для сборки Backend

Q: "Требуются права администратора"
A: Запустите ЗАПУСК.bat от имени администратора (ПКМ → "Запуск от имени администратора")

Q: "Python не найден"
A: Установите Python 3.11+ с https://www.python.org/downloads/

Q: "Не удаётся удалить файлы"
A: Закройте все программы, использующие файлы
