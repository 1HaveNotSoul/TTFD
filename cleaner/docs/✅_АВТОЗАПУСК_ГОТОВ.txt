КАК ИСПОЛЬЗОВАТЬ:
-----------------
1. Откройте TTFD-Cleaner.exe
2. Перейдите на вкладку "Автозапуск"
3. Нажмите "Сканировать"
4. Используйте фильтры для поиска нужных элементов
5. Выберите элемент и нажмите "Включить" или "Отключить"
6. Экспортируйте список в TXT при необходимости

БЕЗОПАСНОСТЬ:
-------------
- Защищённые элементы (красные) нельзя отключить
- LSA Providers и Boot Execute - только для чтения
- Требуются права администратора для изменений

СЛЕДУЮЩИЕ ШАГИ:
---------------
1. Закройте текущий TTFD-Cleaner.exe
2. Скопируйте новый файл из dist\
3. Запустите и протестируйте вкладку "Автозапуск"
4. Проверьте все функции (сканирование, фильтры, включение/отключение)
