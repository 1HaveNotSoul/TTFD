🏗️ АРХИТЕКТУРА:

GUI (Python + Tkinter):
- gui.py - главный файл интерфейса (~600 строк)
- Табы: Очистка, Браузеры, Автозапуск, Приложения, Исключения, Отчёты
- Взаимодействие с CLI через subprocess
- JSON обмен данными
- Асинхронное выполнение (threading)

Backend (C# .NET 8):
- TTFD.Cleaner.Cli.exe - консольное приложение
- Команды: status, scan-cleaning, apply-cleaning, list-startup, etc.
- Безопасная работа с системой
- JSON вывод результатов
- Standalone executable (self-contained)

🚀 КАК ЗАПУСТИТЬ:

1. СБОРКА BACKEND:
   
   Запустите BUILD.bat
   
   Или вручную:
   cd Backend
   dotnet publish -c Release -r win-x64 --self-contained true /p:PublishSingleFile=true
   copy bin\Release\net8.0\win-x64\publish\TTFD.Cleaner.Cli.exe ..

2. ЗАПУСК GUI:
   
   Запустите ЗАПУСК.bat
   
   Или вручную:
   python gui.py
