🚀 КАК ЗАПУСТИТЬ:

1. СБОРКА BACKEND:
   
   Запустите BUILD.bat
   
   Или вручную:
   cd Backend
   dotnet publish -c Release -r win-x64 --self-contained true /p:PublishSingleFile=true
   copy bin\Release\net8.0\win-x64\publish\TTFD.Cleaner.Cli.exe ..

2. ЗАПУСК GUI:
   
   python gui.py
   
   Или двойной клик на ЗАПУСК.bat

📊 CLI КОМАНДЫ:

✅ status                    - Информация о системе
✅ scan-cleaning             - Сканирование (dry-run)
✅ apply-cleaning            - Применение очистки
✅ list-startup              - Список автозапуска
✅ set-startup               - Включение/отключение
✅ list-apps                 - Список приложений
✅ remove-uwp                - Удаление UWP app
✅ export-baseline           - Экспорт baseline
✅ restore-baseline          - Восстановление baseline

Примеры:

TTFD.Cleaner.Cli.exe status
TTFD.Cleaner.Cli.exe scan-cleaning --categories temp,cache
TTFD.Cleaner.Cli.exe apply-cleaning --categories temp --yes
TTFD.Cleaner.Cli.exe list-startup
TTFD.Cleaner.Cli.exe set-startup --id "MyApp" --enabled false
