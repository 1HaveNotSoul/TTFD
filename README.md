# 🎮 TTFD - Макет 1

Минималистичный сайт с нуля без регистрации.

## 🚀 Запуск

```bash
python app.py
```

Сайт откроется на http://localhost:5000

## 📁 Структура

```
макет1/
├── app.py              # Flask приложение
├── templates/          # HTML шаблоны
│   ├── base.html      # Базовый шаблон
│   ├── index.html     # Главная страница
│   ├── settings.html  # Настройки
│   └── profile.html   # Профиль
├── static/
│   ├── css/
│   │   └── style.css  # Основные стили
│   └── js/
│       ├── main.js    # Основной JS
│       └── settings.js # JS для настроек
└── README.md          # Этот файл
```

## ✨ Функции

### Главная страница
- Приветствие пользователя
- Статистика (уровень, очки, ранг, достижения)
- Последняя активность

### Профиль
- Информация о пользователе
- Статистика
- Достижения

### Настройки
- ✅ Редактирование имени и био
- ✅ Загрузка аватарки
- ✅ Выбор цветовой темы (4 варианта)
- ✅ Настройки приватности
- ✅ Настройки уведомлений
- ✅ Все настройки сохраняются в localStorage

## 🎨 Дизайн

- Минималистичный и чистый
- Адаптивный (работает на мобильных)
- Плавные анимации
- 4 цветовые темы
- Без кэша (для разработки)

## 🔧 Технологии

- **Backend**: Flask (Python)
- **Frontend**: HTML, CSS, JavaScript
- **Стили**: Чистый CSS (без фреймворков)
- **Данные**: Тестовые данные (без БД)

## 📝 Особенности

- Нет регистрации/авторизации
- Автоматический тестовый пользователь
- Все изменения сохраняются локально (localStorage)
- Отключён кэш для удобной разработки
- Debug режим включён
